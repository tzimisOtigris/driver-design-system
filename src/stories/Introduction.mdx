# Smart Cities Driver App — Design System

This Storybook documents the component library for the **Smart Cities Driver App**, a delivery task management interface designed for drivers operating in the Athens metropolitan area.

## Design Tokens

| Token | Value |
|-------|-------|
| Primary Blue | `#3a5fff` |
| Primary Green | `#2ecc71` |
| Primary Orange | `#ff9834` |
| Primary Purple | `#8a60ff` |
| Neutral Dark | `#232323` |
| Neutral Medium | `#8a8da5` |
| Neutral Light | `#b0b0b0` |
| Background Page | `#f8fafb` |
| Border Default | `#eceef6` |

## Typography

All text uses **Inter** font family with weights: Regular (400), Medium (500), SemiBold (600), Bold (700).

## Components

### Atoms
- **Button** — Primary, Secondary, Ghost variants with 3 sizes
- **Badge** — Active, Available, Completed status indicators
- **Input** — Form input with label and focus states
- **Avatar** — Circular initials display
- **Divider** — Horizontal separator
- **Logo** — Driver brand mark
- **SearchBar** — Search input with icon

### Components
- **StatCard** — Dashboard metric card (Total, Active, Available, Completed)
- **TaskCard** — Task item for dashboard and list views
- **StopCard** — Route stop with origin/waypoint variants
- **NavItem** — Sidebar navigation button
- **Dialog** — Confirmation modal overlay
- **TopBar** — Page header with optional back and search
